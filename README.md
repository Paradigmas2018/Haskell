## 3. Executando

* Adicionar pacotes Externos:

````
  cabal install split && cabal install ansi-terminal
````

De maneira alternativa, pode-se utilizar o comando **make install**:

* Instando Dependências externas:

````
  make install
````

* Criar executável:

````
  make
````

* Executar:

````
  make run
````

* Excluir executáveis:

````
  make clean
````
